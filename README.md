# kotlin-object-mock-sample

Kotlin の object を Mockito + PowerMock でモックテストするサンプルプロジェクトです。

詳しくは、Qiita の記事 [Kotlin の Object を Mockito + PowerMock でモックする](http://qiita.com/pine613/items/a9e619ce4cb51b729655) をご覧ください。

## 必要なもの

- Java 7 ~

## 動かしかた
### 実行

```
$ ./gradlew run
```

### テスト

```
$ ./gradlew test
```

## ライセンス
Public Domain
